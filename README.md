# 🧩 DayFrame: 멀티플랫폼 위젯 애플리케이션
> **DayFrame은 다양한 일상 도구들을 위젯으로 구성하고, 플랫폼 제약 없이 나만의 보드를 자유롭게 디자인할 수 있도록 돕는 멀티플랫폼 애플리케이션입니다**
>
> 사용자의 위치 기반으로 주차 가능 공간을 빠르게 안내하고, 혼잡도를 시각화하여 보다 효율적인 주차를 도와줍니다.
